# minikit-samples

a collection of hyperpop-ish drums, oneshots, and loops i found by going around twitter/discord and finding mediafires and google drives to custom-made minikits by tracey brakes, goji, bio, angeljelly, trndytrndy, etc 

to use this inside of strudel, enter the following line in your strudel pattern:
```
samples('github:kyrsive/minikit-samples')
```
